Before you download

Free + open source · MIT licensed. macOS will say "can't be opened" the first time because Apple charges $99/year for a Developer ID and this project is free.

  1. Drag Whisp.app to Applications.
  2. Open it once, macOS pops "can't be opened", then hit Done.
  3. Open System Settings -> Privacy & Security.
  4. Scroll down, choose Open Anyway.
  5. Reopen Whisp, confirm Open, and you are done.

You only do this once. Future updates open normally.
